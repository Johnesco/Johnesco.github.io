About This Resume System

This is a dynamic, JavaScript-powered resume website. All resume content is stored in a single JSON data file and rendered dynamically, allowing for multiple output formats and URL-based filtering.

Key Features

Single Source of Truth

All resume data lives in one JSON file. Update once, changes appear everywhere.

Multiple Formats

Styled HTML, plain text (ATS-friendly), visual timeline, and Bootstrap versions.

URL Filtering

Filter by years, industry tags, or skills using URL query parameters.

Customization UI

Interactive interface to build custom filtered resume URLs.

Available Pages

Page Purpose
index.html Main styled resume with modern CSS layout
plaintextresume.html Plain text version optimized for ATS systems and copy/paste
customize.html Interactive filter builder to create custom resume URLs
workhistory.html Visual timeline showing career progression
geeksiresume.html Bootstrap-styled alternative layout

URL Query Parameters

Filter the resume dynamically by adding query parameters to the URL:

Parameter Description Example
years Show only jobs within the last N years ?years=10
tags Filter by industry/role tags (comma-separated) ?tags=healthcare,government
skills Filter which skill categories to display ?skills=testing,tools

Example URLs

  • Last 10 years only: index.html?years=10
  • Healthcare experience: index.html?tags=healthcare
  • Government and healthcare: index.html?tags=government,healthcare
  • Combined filters: index.html?years=15&tags=lead

Technical Details

  • No Build Process: Static files, edit and refresh
  • Vanilla JavaScript: No framework dependencies (except Bootstrap version uses jQuery)
  • CSS Custom Properties: Colors and spacing defined as variables
  • Print Optimized: Includes print styles for PDF generation
  • Default Years Filter: Shows last 15 years of work history by default

Data Format

The resume data follows a modified JSON Resume schema with these sections:

  • basics - Name, title, contact info, summary
  • work - Array of job objects with tags for filtering
  • education - Array of educational institutions
  • skills - Array of skill categories with keywords
  • languages - Array of spoken languages

Updating Content

To update resume content:

  1. Edit js/resumeJSON.js
  2. All pages automatically reflect changes on next load
  3. No compilation or build step required
