Slot Game Terminology

Hold & Spin
Bonus feature where reels lock and respin for additional prizes
Cash Balls / Cash-on-Reels
Credit values that appear directly on reel symbols
Free Games / Free Spins
Bonus rounds with no cost to player
Progressive Jackpots
Jackpots that grow until won
Symbol Transformations
Symbols that change during play
Expanding Reels
Reels that grow to add more win opportunities

What "Game Integration Tester II" Means

"Game Integration Tester"

Not just "find bugs." Integration testing validates that all pieces work together:

  • Game software + physical cabinet hardware
  • Game math + visual presentation
  • Game features + backend systems
  • New builds + existing stable features
  • Documentation + actual implementation

"II" (Level 2)

  • Mid-level, not entry-level
  • Work independently without constant supervision
  • May mentor or support Level I testers
  • Contribute to process improvement
  • Handle complexity and make judgment calls
Think of it like your Apptronik role: Embedded with engineering, translating observed behaviors into actionable documentation, establishing SOPs, bridging operations and engineering. Same concept, but for slot games on physical cabinets.
